## 🎯 实现功能

程序主要实现以下功能：

### ✅ 1. 从文件加载数据（可选）

* 从 `students.txt` 读取数据
* 每行格式：

```text
姓名 年龄 成绩
```

* 自动解析并存入 `vector<Student>`
* 程序加载初期会询问用户是否从`是否从文件加载？ (Y/N，默认 Y)`

### ✅ 2. 显示最后3条数据

* 如果文件有数据：

  * 显示总条数
  * 输出最后3条记录（或不足3条全部输出）

### ✅ 3. 输入学生信息

支持用户手动输入：

* 姓名（长度限制）
* 年龄（0~150）
* 成绩（0~100）

支持：

```text
输入 exit 终止输入
```

### ✅ 4. 数据保存

* 新输入数据：

  * 存入内存（vector）
  * 追加写入文件（append模式）

### ✅ 5. 表格格式输出

使用 `iomanip` 实现对齐输出：

```text
姓名                年龄      成绩
Tom                 20       88.50
```

## 🧠 程序整体结构（重点）

### 🔹 模块划分

该流程为优化版代码流程:

```text
main()
```

## ⚠️ 常见易错点（非常重要）

### ❗ 1. size_t 与负数问题

```cpp
int start = max(0, (int)temp.size() - 3);
```

📌 原因：

* `size_t` 是无符号类型
* 负数会变成超大正数 → 越界

### ❗ 2. getline + cin 混用问题

错误写法：

```cpp
cin >> age;
getline(cin, name); // 可能读到空行
```

✅ 解决方案：
👉 全部使用 `getline`

### ❗ 3. stoi / stof 异常

```cpp
stoi("abc"); // 会抛异常
```

✅ 必须用：

```cpp
try { ... } catch (...) {}
```

### ❗ 4. 文件打开失败未处理

```cpp
if (!file.is_open())
```

👉 必须检查，否则程序异常

### ❗ 5. 空字符串访问越界

```cpp
selected[0]  // 如果 selected == ""
```

✅ 正确：

```cpp
char c = selected.empty() ? 'y' : selected[0];
```

### ❗ 6. 输入未校验

例如：

```cpp
年龄 < 0 或 > 150
成绩 < 0 或 > 100
```

👉 必须手动判断
