## 🧩 本章节涵盖内容

### 1️⃣ 传值调用 vs 引用调用（函数基础）

示例代码展示了两种常见的函数参数传递方式：

* **传值（Call by Value）**

  * 函数接收的是变量的副本
  * 在函数内部修改不会影响原变量

* **引用（Call by Reference）**

  * 函数直接操作原变量
  * 修改会影响外部变量

📌 适合理解：

* 函数参数传递机制
* C++ 中引用 `&` 的作用

---

### 2️⃣ string 字符串基础

使用 `std::string` 进行字符串操作，包括：

* 字符串定义与输出
* 修改字符串中的字符
* 获取字符串长度
* 查找子字符串（`find`）

📌 相比 C 语言的 `char*`：

* 更安全
* 更易用（推荐使用）

---

### 3️⃣ 字符输入处理（cin.get）

通过逐字符读取输入：

* 使用 `cin.get()` 读取单个字符
* 手动拼接字符串
* 判断换行结束输入

📌 适合理解：

* 输入流的底层机制
* 字符级处理方式

---

### 4️⃣ C++ 基本数据类型

示例涵盖常见基础类型：

| 类型 | 说明 |
| ------------------------ | ----------------- |
| int | 整数 |
| short / long / long long | 不同范围整数 |
| float / double | 浮点数 |
| char | 单个字符 |
| bool | 布尔值（true / false） |

📌 帮助理解：

* 不同类型的用途
* 基本类型的区别

---

### 5️⃣ 数组（Array）

演示了：

* 固定大小数组定义
* 使用 `for` 循环访问元素

📌 注意：

* 数组长度固定
* 越界不会报错（但很危险！）

---

### 6️⃣ 指针（Pointer）

基础指针用法：

* 获取变量地址 `&`
* 定义指针 `*`
* 通过指针访问和修改变量

📌 重点理解：

* 内存地址概念
* 解引用 `*p`

---

### 7️⃣ 引用（Reference）

引用是变量的“别名”：

* 使用 `int &ref = x;`
* 修改引用即修改原变量

📌 和指针的区别：

* 更安全
* 语法更简单

## 🚀 如何运行代码

使用 g++ 编译：

```bash
g++ main.cpp -o main
./main
```

如果想运行其他示例函数（如 `main_string`），可以：

* 临时将其改名为 `main`
* 或在主函数中调用它
